# 22 Bean Validation: Advanced Topics

This chapter describes how to create custom constraints, custom
validator messages, and constraint groups using the Java API for
JavaBeans Validation (Bean Validation).

The following topics are addressed here:

  - [Creating Custom Constraints](bean-validation-advanced001.htm#GKFGX)

  - [Customizing Validator
    Messages](bean-validation-advanced002.htm#GKAHI)

  - [Grouping Constraints](bean-validation-advanced003.htm#GKAGV)

  - [Using Method Constraints in Type
    Hierarchies](bean-validation-advanced004.htm#CIHGJBGI)
