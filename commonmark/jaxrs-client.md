# 30 Accessing REST Resources with the JAX-RS Client API

This chapter describes the JAX-RS Client API and includes examples of
how to access REST resources using the Java programming language.

JAX-RS provides a client API for accessing REST resources from other
Java applications.

The following topics are addressed here:

  - [Overview of the Client API](jaxrs-client001.htm#BABBIHEJ)

  - [Using the Client API in the JAX-RS Example
    Applications](jaxrs-client002.htm#BABJCIJC)

  - [Advanced Features of the Client API](jaxrs-client003.htm#BABCDDGH)
