# Persistence

Part VIII explores the Java Persistence API. This part contains the
following chapters:

  - [Chapter 37, "Introduction to the Java Persistence
    API"](persistence-intro.htm#BNBPZ)

  - [Chapter 38, "Running the Persistence
    Examples"](persistence-basicexamples.htm#GIJST)

  - [Chapter 39, "The Java Persistence Query
    Language"](persistence-querylanguage.htm#BNBTG)

  - [Chapter 40, "Using the Criteria API to Create
    Queries"](persistence-criteria.htm#GJITV)

  - [Chapter 41, "Creating and Using String-Based Criteria
    Queries"](persistence-string-queries.htm#GKJIQ)

  - [Chapter 42, "Controlling Concurrent Access to Entity Data with
    Locking"](persistence-locking.htm#GKJJF)
