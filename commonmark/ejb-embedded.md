# 35 Using the Embedded Enterprise Bean Container

This chapter demonstrates how to use the embedded enterprise bean
container to run enterprise bean applications in the Java SE
environment, outside of a Java EE server.

The following topics are addressed here:

  - [Overview of the Embedded Enterprise Bean
    Container](ejb-embedded001.htm#GKFAE)

  - [Developing Embeddable Enterprise Bean
    Applications](ejb-embedded002.htm#GKCRR)

  - [The standalone Example Application](ejb-embedded003.htm#GKCPV)
