# 52 Resource Adapters and Contracts

This chapter examines resource adapters and explains how communications
between Java EE servers and EIS systems are mediated by them.

The following topics are addressed here:

  - [What Is a Resource Adapter?](resources001.htm#GIPGL)

  - [Metadata Annotations](resources002.htm#GIRDD)

  - [Common Client Interface](resources003.htm#GIPJU)

  - [Using Resource Adapters with Contexts and Dependency Injection for
    Java EE (CDI)](resources004.htm#CHDJFIGB)
