# 46 Java Message Service Examples

This chapter provides examples that show how to use the JMS API in
various kinds of Java EE applications.

The following topics are addressed here:

  - [Building and Running Java Message Service
    Examples](jms-examples001.htm#A1251921)

  - [Overview of the JMS Examples](jms-examples002.htm#BABEFBHJ)

  - [Writing Simple JMS Applications](jms-examples003.htm#BNCFA)

  - [Writing More Advanced JMS Applications](jms-examples004.htm#GIWFH)

  - [Writing High Performance and Scalable JMS
    Applications](jms-examples005.htm#BABGEFHC)

  - [Sending and Receiving Messages Using a Simple Web
    Application](jms-examples006.htm#BABBABFC)

  - [Receiving Messages Asynchronously Using a Message-Driven
    Bean](jms-examples007.htm#BNBPK)

  - [Sending Messages from a Session Bean to an
    MDB](jms-examples008.htm#BNCGW)

  - [Using an Entity to Join Messages from Two
    MDBs](jms-examples009.htm#BNCHF)

  - [Using NetBeans IDE to Create JMS
    Resources](jms-examples010.htm#BABDFDJC)
