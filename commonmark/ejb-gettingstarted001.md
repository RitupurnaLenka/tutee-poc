# 33.1 Starting With Enterprise Beans

Here's an overview of the steps you'll follow:

1.  Create the enterprise bean: `ConverterBean`.

2.  Create the web client.

3.  Deploy `converter` onto the server.

4.  Using a browser, run the web client.

Before proceeding, make sure that you've done the following:

  - Read [Chapter 1, "Overview"](overview.htm#BNAAW)

  - Become familiar with enterprise beans (see [Chapter 32, "Enterprise
    Beans"](ejb-intro.htm#GIJSZ))

  - Started the server (see [Starting and Stopping GlassFish
    Server](usingexamples002.htm#BNADI))

The following topics are addressed here:

  - [Creating the Enterprise Bean](ejb-gettingstarted002.htm#GIPSS)

  - [Modifying the Java EE Application](ejb-gettingstarted003.htm#GIPTI)
