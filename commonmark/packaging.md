# 5 Packaging

This chapter describes packaging. A Java EE application is packaged into
one or more standard units for deployment to any Java EE
platform–compliant system. Each unit contains a functional component
or components, such as an enterprise bean, web page, servlet, or applet,
and an optional deployment descriptor that describes its content.

The following topics are addressed here:

  - [Packaging Applications](packaging001.htm#BCGDJDFB)

  - [Packaging Enterprise Beans](packaging002.htm#BCGECBIJ)

  - [Packaging Web Archives](packaging003.htm#BCGHAHGD)

  - [Packaging Resource Adapter Archives](packaging004.htm#BCGDHBHJ)
