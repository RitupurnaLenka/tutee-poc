# 48 Getting Started Securing Web Applications

This chapter describes in greater detail the ways to implement security
for Java EE web applications discussed in a general way in [Securing
Containers](security-intro003.htm#BNBXE). The detail and examples in
this chapter explore these security services as they relate to web
components.

The following topics are addressed here:

  - [Overview of Web Application
    Security](security-webtier001.htm#BNCAT)

  - [Securing Web Applications](security-webtier002.htm#GKBAA)

  - [Using Programmatic Security with Web
    Applications](security-webtier003.htm#GJIIE)

  - [Examples: Securing Web Applications](security-webtier004.htm#BNCBX)
